# Minimal Image Gallery

A clean, responsive image gallery application with hover effects, built with HTML, CSS, and JavaScript.

## Features

- **Responsive Grid Layout**: Adapts to all screen sizes with a flexible CSS Grid layout
- **Interactive Hover Effects**:
  - Smooth image zoom on hover
  - Fade-in captions with upward animation
  - Subtle card elevation
- **Image Filtering**: Filter images by category (Nature, Architecture, Abstract)
- **Clean & Minimal Design**: Simple color scheme with ample whitespace
- **Image Captions**: Descriptive text appears on hover
- **Category Icons**: Visual indicators for each image category

## Technologies Used

- **HTML5**: For structure and content
- **CSS3**: For styling, animations, and responsive design
  - CSS Grid for layout
  - Flexbox for alignment
  - CSS transitions and animations
- **JavaScript**: For interactive filtering functionality
- **Font Awesome**: For icons
- **Google Fonts**: Using Inter font family
- **Unsplash**: For high-quality images
